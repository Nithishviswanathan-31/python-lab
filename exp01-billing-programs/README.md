## Exp 01 — Billing programs

Programs using arithmetic operations, conditional logic, and formatted output.

| File | Description |
|------|-------------|
| 01a_electricity_billing.py | Calculates electricity bill based on units consumed using slab rates |
| 01b_retail_shop_billing.py | Generates itemised bill for retail shop purchases |
| 01c_weight_of_steel_bar.py | Computes weight of a steel bar given length, breadth, and thickness |
| 01d_electrical_current_3phase_ac.py | Calculates current in a 3-phase AC circuit using I = P / (√3 × V × PF) |

## Concepts used
- Arithmetic operators
- if-elif-else conditional statements
- Formatted string output using f-strings
- Math module (sqrt) for 3-phase AC calculation
