## Exp 02 — Simple statements

Programs demonstrating variable manipulation and basic mathematical operations.

| File | Description |
|------|-------------|
| 02a_swap_two_variables.py | Exchanges values of two variables (with and without temp variable) |
| 02b_circulate_n_variables.py | Circulates values of N variables in a list by one position |
| 02c_distance_between_two_points.py | Calculates Euclidean distance between two 2D points |

## Concepts used
- Variable assignment and swapping
- List indexing and slicing for circulation
- Math formula: distance = √((x2−x1)² + (y2−y1)²)
- math.sqrt() and ** operator

## Sample output

**02a — Swap two variables**
```
Before swap: a = 10, b = 20
After swap : a = 20, b = 10
```

**02b — Circulate N variables**
```
Original  : [10, 20, 30, 40, 50]
After circ: [50, 10, 20, 30, 40]
```

**02c — Distance between two points**
```
Point 1: (2, 3)
Point 2: (6, 7)
Distance: 5.66 units
```
