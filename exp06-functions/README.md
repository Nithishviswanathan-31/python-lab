## Exp 06 — Functions

Programs demonstrating user-defined functions, recursion, and modular programming.

| File | Description |
|------|-------------|
| 06a_factorial.py | Calculates factorial using both iterative and recursive approaches |
| 06b_largest_number_in_list.py | Finds the largest number in a list using a user-defined function |
| 06c_area_of_shapes.py | Computes area of circle, rectangle, and triangle using separate functions |

## Concepts used
- Function definition with def keyword
- Recursion (base case and recursive case)
- Passing arguments and returning values
- math.pi for circle area
