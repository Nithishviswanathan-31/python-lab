## Exp 07 — String handling

Programs demonstrating Python's string operations and built-in methods.

| File | Description |
|------|-------------|
| 07a_reverse_string.py | Reverses a given string using slicing and loop methods |
| 07b_palindrome_check.py | Checks whether a given string is a palindrome |
| 07c_count_characters.py | Counts vowels, consonants, digits, and spaces in a string |
| 07d_replace_characters.py | Replaces specified characters or substrings in a string |

## Concepts used
- String slicing: s[::-1]
- String methods: lower(), replace(), count(), isalpha(), isdigit()
- Looping through characters
- Palindrome logic: string == reverse of string
