## Exp 04 — List operations

Programs demonstrating Python list operations using real-world data models.

| File | Description |
|------|-------------|
| 04a_library_items.py | Add, remove, search, and display books present in a library |
| 04b_tappal_car_components.py | Operations on a list of car (tappal) components — insert, delete, sort |

## Concepts used
- List creation, indexing, and slicing
- Built-in methods: append(), remove(), insert(), sort(), reverse(), index()
- Linear search in a list
- Iterating using for loop

## Sample output

**04a — Library items**
```
Library items: ['Python', 'DBMS', 'OS', 'Networks']
After adding  : ['Python', 'DBMS', 'OS', 'Networks', 'AI']
After removing: ['Python', 'DBMS', 'Networks', 'AI']
Search 'DBMS' : Found at index 1
```
